eval函数

eval() 函数用于计算 JavaScript 字符串，并把它作为脚本代码来执行。如果参数是一个表达式，eval() 函数将执行表达式。如果参数是 Javascript 语句，eval() 将执行 Javascript 语句。所以当我们点击按钮，将表达式写入“显示屏“，该函数便会自动执行计算。
